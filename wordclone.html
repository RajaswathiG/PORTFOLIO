Project: Wordle Clone

The Wordle Clone project is a web-based game inspired by the popular word guessing game, where players guess a hidden five-letter word within six attempts.

Technology Stack Used

  • Frontend: HTML, CSS, JavaScript
  • Game Logic: JavaScript for word generation and guessing mechanism
  • Styling: Custom CSS for layout and design

Features

  • Gameplay: Guess a five-letter word in six attempts
  • Feedback: Visual cues for correct letters and positions
  • Responsive Design: Adapts to different screen sizes

Outcome

The project provides an interactive and engaging game experience similar to Wordle, challenging players' word-guessing skills.
