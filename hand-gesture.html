Hand Gesture Controlled Wheelchair Project

The "Hand Gesture Controlled Wheelchair" project aims to provide enhanced mobility and independence for paralyzed individuals by allowing them to control the wheelchair's movements using intuitive hand gestures.

Components Used

  • Motion Sensor: MPU6050 for detecting hand gestures
  • Microcontroller: ESP32 for processing gesture commands
  • Wireless Protocol: ESPNOW for reliable data transmission
  • Wheelchair Control: Motor driver to translate gestures into wheelchair movements
  • User Interface: Feedback through LCD display for real-time interaction

Features

  • Gesture Recognition: Detects and interprets hand movements for precise control
  • Accessibility: Improves mobility and independence for paralyzed individuals
  • Safety: Built-in safety features such as emergency stop and obstacle detection
  • User-Friendly: Intuitive interface with minimal physical effort required

Outcome

The project successfully demonstrates a viable solution for enhancing the quality of life for paralyzed individuals through innovative technology, providing reliable and responsive control of the wheelchair.

Future Enhancements

Potential future enhancements include integrating voice commands for additional control options, improving gesture recognition accuracy, and expanding compatibility with different wheelchair models to reach a broader user base.
